Interrogative Sentences

Esperanto uses a great system to ask and answer questions. this system is called correlatives. Any given word is created by combining one of 5 prefixes and 9 roots. Using a system creates consistency and ease of learning. Yes / no questions are made with a "ĉu" at the beginning of the sentence. Literally this means whether, but contextually is more like "is". For example, "The ball is on the table." vs. "Is the ball on the table?". The verb and subject do not trade places in Esperanto to make a question.

Prefix Root
Question k- Individual -iu
Pointer t- Object -io
Indefinite (none) Kind -ia
Universal ĉ- Place -ie
Negative nen- Time -iam
Amount -iom
Manner -iel
Reason -ial
Possession -ies

Individual

  1. kiu - who / which
  2. tiu - that one
  3. iu - someone
  4. ĉiu - everyone
  5. neniu - no one

Object

  1. kio - what
  2. tio - that thing
  3. io - something
  4. ĉio - everything
  5. nenio - nothing

Kind

  1. kia - what kind / which
  2. tia - that kind
  3. ia - some kind
  4. ĉia - every kind
  5. nenia - no kind

Place

  1. kie - where
  2. tie - there
  3. ie - somewhere
  4. ĉie - everywhere
  5. nenie - nowhere

Time

  1. kiam - when
  2. tiam - then
  3. iam - sometime
  4. ĉiam - always
  5. neniam - never

Amount

  1. kiom - how much
  2. tiom - that much
  3. iom - some
  4. ĉiom - all
  5. neniom - none

Manner

  1. kiel - how
  2. tiel - this way
  3. iel - somehow
  4. ĉiel - every way
  5. neniel - now way

Reason

  1. kial - why
  2. tial - because
  3. ial - some reason
  4. ĉial - every reason
  5. nenial - no reason

Possession

  1. kies - whose
  2. ties - that one's
  3. ies - somebody's
  4. ĉies - everybody's
  5. nenies - nobody's
